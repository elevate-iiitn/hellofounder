# Contributing

Thank you for your interest in this project.

## Contribution Policy

This repository contains a **private, client-funded project** maintained by an internal team.

At this time:

- ❌ We do **not** accept external contributions
- ❌ Pull requests from outside the core team will not be reviewed
- ❌ Feature requests from the public are not being considered

## Authorized Contributors

Only approved team members and collaborators explicitly added to this repository
are permitted to:

- Open issues
- Submit pull requests
- Modify code or documentation

## Questions or Inquiries

If you believe you have reached this repository in error or have a legitimate
business inquiry, please contact the project maintainer directly.
